EAR SLURM plug-in

EAR SLURM plugin allows to dynamically load the EAR Loader for the SLURM jobs (and steps), if the flag --ear=on is set or if it is enabled by default. The Loader will be executed at each job step, intercepting all MPI calls and passing this information to the EAR Library.

Configuration

Visit the SLURM SPANK plugin section on the configuration page to set up properly the SLURM /etc/slurm/plugstack.conf file.

You can find the complete list of EAR SLURM plugin accpeted parameters in the user guide.
